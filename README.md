Voronoi++
=======

Voronoi++ is a C++ library to calculate voronoi diagrams for a given set of points.

Content
------------

 * [Requirements](#requirements)
 * [Building](#building)
 * [License](#license)

Requirements
---------------------

 * [gcc 4.6](http://gcc.gnu.org/)
 * [CMake 2.6](http://www.cmake.org/)
 
 Building
------------------------------------------

    > mkdir build && cd build
    > cmake ..
    > make
